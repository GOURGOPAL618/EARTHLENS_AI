# Code of Conduct

## Our Standard

This is an open, respectful project. Everyone contributing or engaging
with EarthLens AI is expected to:

- Be respectful and constructive in all communication
- Welcome feedback and different perspectives
- Focus on what is best for the project and the community

## Unacceptable Behavior

- Harassment, insults, or discriminatory language
- Personal attacks in issues or pull requests
- Deliberately disruptive behavior

## Enforcement

Violations can be reported to the author directly via GitHub.
Maintainers reserve the right to remove comments or ban contributors
who violate this standard.
