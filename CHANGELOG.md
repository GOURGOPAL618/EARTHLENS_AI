## [0.1.0] — 2025

### Added
- NDVI Analysis module with 5-class vegetation classification
- Water Detection using NDWI + MNDWI dual-index masking
- Land Use Classification — Random Forest, Decision Tree, Logistic Regression
- Change Detection — NDVI differencing + Change Vector Analysis
- Burn Area Detection — NBR, dNBR, USGS severity classes
- Urban Expansion — NDBI, UI, IBI indices
- Synthetic Sentinel-2 band generator (offline mode)
- Streamlit dashboard with Folium maps and Plotly charts
- Jupyter notebook for NDVI ML experiments
